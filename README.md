# 🤖 BotCodex - 机器人学个人知识库与代码实践

这是一个持续更新的机器人学学习仓库，旨在系统化地记录从理论到实践的完整路径。内容涵盖机器人运动学、动力学、控制、感知（如SLAM）等核心领域，包含详尽的阅读笔记、算法推导、仿真与实战代码。

**主要板块**

📚 **理论笔记**: 经典教材与论文的精读摘要、核心公式推导。

💻 **代码实践**: ROS/Gazebo、MATLAB及Python实现的关键算法Demo。

🗺️ **项目轨迹**: 小型实验项目与学习路径的完整记录。

作为一名从软件切入机器人学的开发者, 我希望通过这个仓库沉淀知识, 并与社区交流。欢迎 Star ⭐ 和 Fork, 共同构建!

```
/BotCodex
├── 📖 theory/          # 理论笔记(按领域分子目录)
├── 💻 src/             # 源代码(按项目或算法分子目录)
├── 📁 docs/            # 项目文档、教程
├── 🗓️ projects/        # 综合项目记录
```
